# SQUAD Plugin Development

This lab teaches you to build and publish plugins that extend SQUAD capabilities.

## Structure
- `plugins/jira-integration` - Jira sync plugin
- `plugins/db-migration-checker` - Database migration safety checker
- `plugins/changelog-generator` - Changelog generator plugin
- `test-project/` - Sample project for testing plugins

See [APPMODLAB.md](APPMODLAB.md) for complete lab documentation.
